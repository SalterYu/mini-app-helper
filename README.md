自动创建小程序目录渐进式工具，暂时绑定jgb模板
===

####安装方法

```vue
npm i -g mini-app-helper
```

####使用方法

```vue
thmp cp <dir>  // 生成page目录
thmp cc <dir>  // 生成component目录
```

####配置
```vue
-t, --target [target]', 'target app, default: weapp, type: weapp || baiduapp
-f, --filename [name]', 'target filename, default: index
```
